FAQ

Frequently Asked Questions

What is this website about?

This website is a demo project showing form validation, sorting, and an FAQ section using only HTML, CSS, and JavaScript.

Do I need JavaScript for this page?

No! This page uses only HTML and CSS. The FAQ accordion is built using the <details> and <summary> elements.

Is this page mobile responsive?

Yes, the layout adapts to various screen sizes using CSS Flexbox and media queries.

Where can I find the source code?

You can structure it using the project-root directory as shown and reuse the shared style.css.
